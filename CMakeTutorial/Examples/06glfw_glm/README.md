# 06glfw_glm

this is simple framework for drawing

## Prerequisites
  * C++14 compiler
  * CMake (3.5 or latter)  
  * Boost (1.60 or latter)  
  * glm (0.9.8.4 or latter)  
  * glfw (3.2 or latter)  
  * pkg-config (if UNIX)  

### Build
#### Win10: 
PS\\> .\build.bat
#### Linux:
[user]$ bash build.sh
